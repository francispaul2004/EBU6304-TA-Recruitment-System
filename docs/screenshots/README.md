# Screenshot Placeholder Guide

Place captured UI screenshots in this folder using the naming pattern in `../User-Manual.md`.

Recommended minimum set:

- `P01_Login.png` ... `P13_Admin_Dashboard.png`
- `D01_Validation_Error.png` ... `D05_Match_Workload_Warning.png`

For final submission, ensure each main frame (P01-P13) has at least one screenshot.
